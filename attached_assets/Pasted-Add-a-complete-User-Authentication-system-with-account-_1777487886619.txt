## 👤 USER AUTHENTICATION SYSTEM

### Auth Context
Create `src/hooks/use-auth.tsx` with:
- AuthProvider wrapping the entire app (inside LanguageProvider)
- State: `{ user, isLoading, isAuthenticated }`
- User object: `{ id, name, email, phone, avatar, bookings[], enrolledCourses[], createdAt }`
- Persist logged-in user to localStorage key `salam_user`
- Functions: `login()`, `register()`, `logout()`, `updateProfile()`

### Auth Flow (NO backend — simulate with localStorage)
- On `register()`: save user to localStorage `salam_users` array, auto-login
- On `login()`: find user in `salam_users` by email+password, set as current user
- On `logout()`: clear current user, redirect to home
- Passwords stored as plain string in localStorage (this is a prototype — note in comments)

## 🎨 MODAL/POPUP DESIGN SYSTEM

All modals share these rules:
```css
/* Overlay */
backdrop: rgba(45, 74, 69, 0.5) with blur(4px)

/* Modal container */
background: rgba(255, 255, 255, 0.95)
backdrop-filter: blur(20px)
border: 1px solid rgba(127, 169, 155, 0.3)
border-radius: 24px
box-shadow: 0 25px 50px rgba(90, 138, 128, 0.2)
max-width: 420px
width: 90vw

/* Animations */
- Enter: scale(0.9) opacity(0) → scale(1) opacity(1), 300ms ease-out
- Exit: scale(0.95) opacity(1) → scale(0.9) opacity(0), 200ms ease-in
- Overlay: opacity(0) → opacity(1), 200ms

/* Input fields */
border: 1.5px solid var(--sage-muted)
border-radius: 12px
padding: 12px 16px
focus: border-color var(--sage), box-shadow 0 0 0 3px rgba(127,169,155,0.15)

/* Primary button */
background: var(--sage)
color: white
border-radius: 50px
padding: 12px 32px
hover: background var(--sage-dark), transform translateY(-1px)
```

## ⚙️ TECHNICAL NOTES

- Use Write() to rewrite entire files — do NOT use Update()/str_replace to avoid edit loop errors
- Create a reusable `<Modal>` base component in `src/components/ui/modal.tsx`
- Use React Portal (`ReactDOM.createPortal`) to render modals at document.body level
- Trap focus inside open modals (accessibility)
- Close modal on: Escape key, clicking overlay background
- Prevent body scroll when modal is open (`overflow: hidden` on body)
- All modal text must be bilingual using the existing useLanguage hook
- Connect AuthContext to Sessions page: if user is logged in, pre-fill booking form with their name/email/phone
- Connect AuthContext to Courses page: show "استمري / Continue" instead of "اشتركي / Enroll" for enrolled courses
- Always use Write() for full file rewrites

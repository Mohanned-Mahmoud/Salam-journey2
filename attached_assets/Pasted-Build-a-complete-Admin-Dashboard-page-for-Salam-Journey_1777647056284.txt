## ⚙️ TECHNICAL REQUIREMENTS

- Route: `/admin` — add to App.tsx routes
- All data operations on localStorage
- No page reload needed — all updates reactive with React state
- Admin page does NOT use the main Navbar/Footer
- Use Write() for all file changes
- Modals use the same glass-card design from the main site
- All text bilingual (admin can be English-only is fine for simplicity)
- Sidebar uses `dir="rtl"` Arabic labels

## DATA CONNECTIONS:
- Courses admin → `salam_courses` localStorage → Courses page reads from same key
- Products admin → `salam_products` localStorage → Products page reads from same key  
- Testimonials admin → `salam_testimonials` localStorage → Homepage carousel reads from same key
- Bookings admin → reads `salam_bookings` + `bookedSlots` localStorage
- Settings admin → `salam_settings` → Booking calendar reads available times from this
- Users admin → reads `salam_users` localStorage

Build a full Arabic RTL website for "Salam Journey" (رحلة سلام) — an Arabic Islamic parenting & self-awareness platform for mothers, founded by Coach Iman. The site is fully in Arabic, right-to-left layout.

---

## 🎨 Brand & Color Palette (extracted from salamjourney.com)

Use these exact colors as CSS variables:

```css
--color-primary: #C8A97E;        /* warm gold/sand — main brand color */
--color-primary-dark: #A8845A;   /* deeper gold for hover states */
--color-secondary: #E8D5B7;      /* soft cream/beige */
--color-bg: #FAF6F0;             /* off-white warm background */
--color-bg-dark: #2C2420;        /* deep warm brown — for dark sections */
--color-accent: #8B6B47;         /* earth brown accent */
--color-text: #3D2B1F;           /* warm dark brown for body text */
--color-text-light: #7A6055;     /* muted warm text */
--color-white: #FFFFFF;
--color-success: #7DAA80;        /* muted green for CTAs */
```

**Aesthetic**: Warm, calm, organic, feminine — like a cozy Islamic home. Think soft textures, round corners, gentle shadows. NOT purple, NOT tech-y. Inspired by natural earth tones.

---

## 🖋 Typography

- Display/Headers: Use **"Tajawal"** or **"Cairo"** (Arabic Google Fonts) — bold, warm
- Body: **"Noto Naskh Arabic"** or **"Amiri"** for readable Arabic body text
- Load from Google Fonts CDN

## 🔧 Technical Requirements

- **Framework**: React (Vite) or plain HTML/CSS/JS — your choice
- **Direction**: `dir="rtl"` on `<html>`, all layout RTL
- **Language**: All text in Arabic
- **Responsive**: Mobile-first. Works perfectly on phone (most users are on mobile)
- **Navigation**: Sticky top navbar with logo + links + hamburger on mobile
- **Smooth scroll** between sections on homepage
- **Footer**: Links | واتساب | إنستغرام | يوتيوب | حقوق محفوظة لأكاديمية سلام ٢٠٢٤

## 🎯 UX Inspiration (from reference funnels)

- Warm, high-converting funnel style like Arabic coaching sites
- Clear pain→solution flow: show the mother's struggle → the transformation → the offer
- Social proof (testimonials) prominently placed
- Urgency elements: "عدد محدود من الجلسات" / "سارع بالتسجيل"
- WhatsApp CTA floating button (bottom right corner) linking to `https://wa.me/`

---

## 🚫 Do NOT:
- Use purple or generic tech colors
- Use Inter or Roboto fonts
- Make it look like a generic SaaS site
- Use LTR layout

Make it feel like a warm, trusted, feminine Arabic educational platform — the kind an Arab mother would trust and feel comfortable on.
